# BloomTech
 Resource pack adding bloom to objects by using Lumenized
 
 ## Currently supports

  - Vanilla Minecraft
  - Chisel 
  - ExtendedCrafting
  - Rustic
  - Integrateddynamics
  - Thaumcraft
  - Thaumadditions
  - Thaumic augmentation
  - Quark
  - End Reborn
  - Actually Additions
  - Random Things
  - MIA
  - Ice And Fire
  - NuclearCraft / Trinity
